# API Documentation

API documentation is served by Swagger UI at:

- `http://localhost:5001/api/docs` (local development)
- `<your-backend-url>/api/docs` (deployed environment)

Start the backend, then open `/api/docs` to view all endpoints, request/response schemas, and auth requirements.
